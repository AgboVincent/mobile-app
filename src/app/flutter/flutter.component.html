PROJECT ONE

WORLD TIME APP

The app will detect the time for any country at any instant time. It was developed using the world time API. The app will display a day mode if the time for the country is still day time and night mode if the time for the country is night time.

PROJECT TWO

COFFEE ORDERING/REQUEST APP

The user of the app order for a coffee. The request is updated to the cloud firestore database when the request is made. The user can decide the thickness of the coffee wanted. It will get updated to the cloud firestore and also update to the app whenever any changes in coffee requirements are made.

THE SIGN IN PAGE

The user sign in with email and password. This details was used with firebase authentication. Restrictions were put to the login of the user. If expected requirements were not entered correctly in the field, the user gets a notification message.

USER LOGGED IN PAGE

This page is displayed when the user logged in successfully. The app reads the user and determine whether it is a new or an already existing user. The last record is get displayed

THE REGISTRATION PAGE

The user start using the app when registered. The key two detail for registration are email and password. The app uses firebase email and password authentication.

THE SETTINGS PAGE

This is the page where the user will select the type of coffee wanted, the number of sugars to be added to the coffee. The user also selects the thickness of the coffee wanted. The selected data gets updated to the database.

THE DATABASE COLLECTION OF THE USER SELECTIONS IN PLACING AN ORDER

The database constantly changes as the user update their data
